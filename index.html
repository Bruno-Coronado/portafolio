CODIFICAR. DISFRUTAR. REPETIR.

COA busca renovar su marca y con ello su ecosistema digital para la temporada verano 21 – 22, además de potenciar su sitio web con una estructura poco definida de la información, lo que generaba conflictos para la venta de sus productos. Se rescataron elementos gráficos propios ya disponibles pero poco resaltados por la marca, resaltando la identidad en su sitio web y redes sociales.
Otro elemento que logramos fortalecer es el tipo de comunicación usado en las distintas secciones de la web, logrando así conectar de una manera diferente con los usuarios y permitiendo diferenciarse de mejor manera de la competencia.
Parte importante de los cambios realizados fueron a nivel de estructura y jerarquía de las distintas secciones de la web, además de una categorización de los productos que permita una mejor experiencia y navegabilidad de los usuarios aumentando las tasas de permanencia y conversión del sitio. Por último resaltar que en la actualidad se han realizado otras adaptaciones de la web pero parte fundamental del sitio mantiene los fundamentos que permiten generar mejores resultados.

Sitio Web COA SURF

La renovación del sitio web de CNA busca fortalecer las relaciones con sus clientes actuales y proporcionarles una mejor experiencia que vaya mas allá de la venta. Por lo cual se requiere de una web compuesta por distintos niveles de jerarquía y estructuras que permitan una navegación intuitiva que comparta información y aporte valor y dinamismo a la operación de sus clientes, además de contar con un portal que permita conectar las necesidades de cada cliente con las herramientas disponibles de CNA. Es por esto que la mejor opción es trabajar en base a plantillas y plugins proporcionados por Wordpress, lo que permite ser a la web escalable y más eficiente en la gestión del contenido. También es importante mencionar la capacidad responsiva del sitio, lo que permite a usuarios en terreno realizar diferentes tipos de acciones desde dispositivos móviles.

Sitio Web CNA Agro

Proyecto personal basado en HTML5, CSS y JS para el desarrollo de una web responsiva que cumpla con aspectos básicos de diseño como los las secciones; hero, menú desplegable, botones y tarjetas. Lo principal a resaltar en este proyecto es como se utilizan contenedores en CSS para que las distintas secciones se ajusten a diferentes tamaños de pantalla. También en el uso de clases para dar estilo a los diferentes elementos, además de un poco de interacción al momento de interactuar con el elemento desplegable (menú hamburguesa) de la versión web. El código prontamente estará disponible en Github ya que se trata de un proyecto local, lo que no permite ser visualizado en web.

Proyecto Web Responsive

El proyecto consistió en el desarrollo de un sitio web en WordPress, diseñado para ofrecer una doble funcionalidad: comunicar con claridad la propuesta de valor del negocio y, al mismo tiempo, facilitar la venta especializada de productos a través de la integración con WooCommerce. Se trata de una empresa con amplia trayectoria en el sector industrial, enfocada en la comercialización de pinturas, insumos de demarcación vial, construcción y soluciones para la industria. Desde un enfoque técnico, se priorizó la estructura del sitio y la optimización del renderizado, asegurando una navegación rápida, segura y accesible tanto en dispositivos móviles como de escritorio. La experiencia de usuario fue un pilar esencial en el diseño, buscando siempre facilitar el recorrido del visitante hasta la generación de una cotización personalizada. A diferencia de un modelo de e-commerce tradicional orientado al retail, este proyecto se centró en un flujo de conversión adaptado al modelo B2B y B2C especializado, con un sistema simplificado de cotización a través de formularios de contacto integrados con WooCommerce. Esta funcionalidad permite al cliente seleccionar productos de su interés y enviar una solicitud de cotización agrupada, estableciendo un canal de contacto directo con un ejecutivo comercial. Esto genera no solo una instancia de conversación más profunda, sino que también aumenta la capacidad de la empresa para entender las necesidades de sus clientes y mejorar las oportunidades de negociación a través de un trato directo.

Sitio Web Prodquim

Proyecto: Aplicación Web CRUD con TypeScript (Stack MERN) Este proyecto consiste en el desarrollo de una aplicación web CRUD (Crear, Leer, Actualizar, Eliminar) implementada con TypeScript tanto en el backend como en el frontend, utilizando el stack MERN (MongoDB, Express, React, Node). Backend (API REST con Node + Express + MongoDB) Configuración de un servidor en Node.js con TypeScript. Uso de Express para la creación de rutas y controladores REST. Conexión a MongoDB mediante Mongoose para la gestión de datos. Implementación de dependencias clave: Morgan para el monitoreo de peticiones. Dotenv para la gestión de variables de entorno. Cors para la comunicación segura entre frontend y backend. Arquitectura organizada en módulos: modelos, rutas, controladores y configuración de base de datos. Ejecución y compilación automática con ts-node-dev. Frontend (React + TypeScript) Creación del proyecto con Create React App usando la plantilla de TypeScript. Estructuración en componentes reutilizables (ej. Navbar, VideoList, VideoForm). Implementación de enrutamiento con React Router DOM. Integración con la API backend para consumir datos mediante operaciones CRUD. Listado y gestión de videos favoritos almacenados en la base de datos. Formularios para agregar videos con datos descriptivos y vínculos externos (ej. YouTube). Objetivo El sistema permite gestionar una colección de videos favoritos, mostrando en el frontend la información registrada en la base de datos y ofreciendo la posibilidad de reproducirlos directamente desde la aplicación. Este proyecto fue desarrollado como un ejercicio práctico de integración full stack con TypeScript, aplicando buenas prácticas en la organización del código y en la conexión entre frontend y backend.
